Installation

In order to install this theme you must have a working version of WordPress already installed

You can install this theme in 2 ways: via FTP or via WordPress theme upload function.

FTP installation

  • Connect to your hosting via an FTP software
  • Unzip the Corsa.zip file on your computer
  • Upload the extracted Corsa theme folder into wp-content > themes in your wordpress installation

WordPress installation

  • Navigate to Appearance > Themes.
  • Click Install Themes and then press the upload button.
  • Find the Corsa.zip file on your computer and click Install Now button.

After the installation Corsa Theme must appear at Appearance > Themes page, where you can activate it.

After you activate the theme, go to Pages > Add New. Select at Page Attributes area Template:  Home Page, give a reasonable name (for example "Corsa Home") and Publish.

When you activate the theme, proceed to Settings > Reading page, set "Front page displays" option to "A static page (select below)" and select a page, that you have created in previous step. Dont forget to save the changes.

Demo Data Import

You can quickly set up your theme same way as our demo site.

To do so, go to Appearance > Theme Options, General tab and press the "Import Demo Data" Button. Wait until import process is finished and that's it.

 

In case something went wrong with automatic import, follow the steps below:

  • The XML file is included to theme files. It is named demo_data.xml and located in a folder called xml
  • Navigate to WP Admin > Tools > Import
  • Select and install WordPress importer
  • Once its installed, select the XML file and upload it. It will take a few minutes to install and upload all content
  • Now navigate to WP Admin > Appearance > Menus page and set menus to apropriate locations

In order to speed up the import process, or in case of upload fail, leave the "download images and attachments" box unchecked. That will give you all the pages and posts without the attachments. This is a good option because users will need to use their own images anyways.

 

XML file doesn't include images, sliders and advanced theme options settings. See corresponding sections for more information.

Theme Options

Corsa Theme has an advanced Theme Options panel that contains all needed settings. There are too many options to go over in these docs but please take some time and navigate through each tab. Go to Appearance > Theme Options and take a look.

If you have installed Corsa theme for first time, it is recommended to reset Theme Options by using "Options Reset" button at bottom left of Advanced Options Interface

Main Page Arrangement

Corsa Theme Main Page is built with "Page Sections" posts and Main Menu.

In order to build your Main Page first create it's sections: use Page Sections > Add Page Section admin page and create as many sections as you need to represent your service or product.

Then proceed to Appearance > Menus. Create a new menu and set in Theme Locations box as Top Menu. Then choose Page Sections that you want to see at your Home Page and add them to Menu (if you do not see the Page Sections list under Pages list, open the Screen Options in top right corner of Admin Panel and check them to appear).

Order of Page Sections in Top Menu corresponds to the order they appear on your Main Page. So you can arrange them as you want.

After adding all wanted Page Sections and reordering them, save the Menu and check how your new amazing Main Page looks by visiting site.

Portfolio Arrangement

Corsa Theme provides possibility to present your portfolio in a simple way.

First, create your Portfolio Items by using Portfolio > Add Portfolio Item page in Admin area. There you can add Image, Title and description of your work.

Then you can arrange your Portfolio Items by adjusting their Publish Date. You can adjust it at Publish area at Edit page of Portfolio Item. The newer the date, the higher Portfolio Item will be placed in list.

After creating and arranging your Portfolio Items, just create a new Page Section or edit already created one and place [portfolio] shortcode to it's content.

Shortcodes

[siubsection] - building blocks for main page sections and separate pages

When you want highlight some section of your page, apply alternate colors to it or add background image (probably with parallax effect) - this shortcode will be your choice.

Attributes:

  • color - leave blank for default Color style or use one of following values: alternate, primary (white text on primary background), dark (white text on dark background)
  • full_width - 1 or 0 (yes or no respectively) - expand subsection to full width of screen
  • full_height - 1 or 0 (yes or no respectively) - remove vertical indents
  • background - path to background image
  • parallax - apply parallax effect to background image

Example - how our demo's home section is organized:

[subsection color="dark" full_width="0" full_height="0" background="http://us-themes.com/wp/Corsa/wp-content/uploads/img-4.jpg" parallax="1"]

[home_heading]
[heading_line type="white" bold="0" new_line="0"]Wonderful Digital Things[/heading_line]
[heading_line type="white" bold="0" new_line="1"]and a pinch of creativity[/heading_line]
[/home_heading]

[button url="#about" text="Learn More" size="big" type="primary" icon="" external="0"]

[/subsection]

[button]

A link styled as button

Attributes:

  • text
  • url
  • external (optional) - 1 or 0 (yes or no respectively) - if yes, link will open in new Tab
  • type - default, primary or secondary
  • size - small, big or leave blank for normal
  • icon (optional) - FontAwesome Icon name. Full list of icons

Example:

[button url="#about" text="Learn More" size="big" type="primary" icon="" external="0"]

[icon]

Single icon

Attributes:

  • icon - FontAwesome Icon name. Full list of icons
  • color - Color Style - leave blank for default Text color, primary, secondaty, fade or border
  • size - tiny, small, medium, big, huge
  • with_circle - 1 or 0 (yes or no respectively) - if yes, icon will be placed into circle
  • link (optional) - enter an url to make your iconbox a link

Example:

[icon icon="star" color="primary" size="small" with_circle="0" link=""]

[iconbox]

Attributes:

  • iconpos - Icon Position: top or left
  • icon - FontAwesome Icon name. Full list of icons
  • image - Path to 32x32 px image. Image overrides icon setting
  • with_circle - 1 or 0 (yes or no respectively) - if yes, icon will be placed into circle
  • title - iconbox title
  • link (optional) - enter an url to make your iconbox a link

Example:

[iconbox iconpos="left" icon="puzzle-piece" with_circle="1" title="Strategy" link="" img=""]
Creating a design it is not an end in itself, but a means... 
[/iconbox]

Translating Corsa

You can translate Corsa into your language using files located at Corsa/languages folder. In order to translate the theme, follow the steps below:

  • Locate default.po and default.mo files at Corsa/languages folder
  • Edit the .po file using POEdit
  • Use the translation field to make replacements
  • Save file with your language name (File > Save as at program menu), for example: es_ES.po. It will generate both a .po and .mo file for your translation.
  • Go to your server via FTP, and upload the language files to the languages folder of Corsa theme (located at wp-content/themes/Corsa/languages)
  • Edit wp-config.php located in the root folder of wordpress and define the WPLANG constant with your language name which should be similar to the translation file name (in this example 'es_ES')

Corsa is WPML ready

If you need a Multi-Language site, then we recommend using the WPML plugin. You would need to buy a license for it, then install it and set it up. All information and documentation can be found at plugin's website: WPML.org

F.A.Q.

When I try to install the theme, I got the message: The package could not be installed. The theme is missing the style.css stylesheet.

A common issue that can occur with users new to installing WordPress themes is a “Broken theme and/or stylesheets missing” error message being displayed when trying to upload or activate the theme. This error message does not mean that the theme you have purchased is broken, it simply means it has been uploaded incorrectly. Luckily, there is a very easy fix.

I see that the new version of Corsa is avaible. How can i install this new version?

You can download a new version of Corsa at your downloads page. After downloading there are 2 ways of upgrading the theme:

1) Recommended: upload contents of Corsa folder from downloaded archive to wp-content/themes/Corsa folder on your server overwriting all files (via FTP or other file transfer method)

2) If you have no access to the server, or you dont know where the theme folder is, you can go to Appearance > Themes at WP admin, then activate any other Theme (like Twenty Twelve from standard package) and delete current Corsa theme. Don’t worry, all posts, pages and other items will be saved at Database. Then go to Install Themes tab and Upload new .zip file with updated Corsa theme, then activate it. After that please go to Appearance > Theme options and verify that everything is up to date there and check if Menu is selected right (it must be OK, just make sure)
